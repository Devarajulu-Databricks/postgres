Step 1: Access PostgreSQL as a Superuser
If you can connect to PostgreSQL with another superuser role, you can create the postgres role.
Open Command Prompt as Administrator:
Press  Win + X and select "Command Prompt (Admin)" or "Windows PowerShell (Admin)".
Alternatively, search for "cmd" in the Start menu, right-click on "Command Prompt", and select "Run as administrator".
Connect to PostgreSQL with an Existing Superuser Role:
Replace  another_superuser with an existing superuser role.
psql -U another_superuser -d postgres
